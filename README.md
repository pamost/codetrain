# codetrain - тренажёр по языкам программирования

Веб-приложение тренажер для изучения основ языков программирования (поддерживаются Go и JS) с помощью тестов с автопроверкой

## Функциональность
- Создание/редактирование/удаление тестов и вопросов
- Прохождение тестов с множественным выбором
- Автоматическая проверка и отображение результатов с пояснениями
- Адаптивный интерфейс на TailwindCSS

## Установка и запуск
1. Клонируйте репозиторий
2. Создайте виртуальное окружение: `python3 -m venv venv`
3. Активируйте: `source venv/bin/activate` (или `venv\Scripts\activate` на Windows)
4. Установите зависимости: `pip install django`
5. Выполните миграции: `python manage.py migrate`
6. Запустите сервер: `python manage.py runserver`

## Технологии
- Django 6.x
- TailwindCSS
- SQLite
